## How to build tests

### Requirements

* C++17 conformant(ish) compiler. GCC 7 or Clang 5.0 will do.
* CMake 3.1 or newer.

### Build instructions

The usual CMake stuff:
```
mkdir build && cd build
cmake ..
make
```
